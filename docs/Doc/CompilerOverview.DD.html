Compiler Index

The compiler lexes and parses code into Intermediate Codes which are then made into JIT or AOT machine code. The compiler 
preprocesses during lexing, and optimizes after Intermediate Code generation.

For parser reference, see ParseExpression(), ParseStatement().
For Intermediate Code reference, see BackLib, Intermediate Code Attributes, Combining Consts, Choosing Reg Vars.
For machine code backend reference, see IC Struct, COCCompile and OptPass789A(), BackEnd.
