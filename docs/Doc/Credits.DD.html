Credits

Terry A. Davis wrote all of TempleOS over a span of 15.0 years (full-time). ZealOS, written over a span of 2.4 years, is a 
fork of TempleOS.   It can run on some bare metal 64-bit PC's from about 2007-2019 with no layering, libraries, tools, modules 
or anything from other sources. Otherwise, you run it in a virtual machine, like VMware, QEMU or VirtualBox.    It is 
independent and stands alone.  100% of the src code is including on all distro's, from the kernel to the compiler to the boot 
loaders!    It is public domain, not GPL.
